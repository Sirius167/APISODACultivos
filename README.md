📊 DOCUMENTACIÓN DEL PROYECTO: CULTIVOS DE COLOMBIA
🎯 ¿QUÉ ES?
Es una aplicación web que muestra datos agrícolas de Colombia usando la API oficial de datos.gov.co. Muestra información sobre cultivos, producción, rendimiento, etc.

📁 ARCHIVOS DEL PROYECTO
Archivo	Qué contiene
index.html	La estructura de la página (botones, tablas, gráficas)
styles.css	Todos los colores, fuentes y diseño visual
app.js	La lógica que conecta con la API y maneja los datos
🛠️ TECNOLOGÍAS USADAS
- HTML5 - Para la estructura

- CSS3 - Para los estilos

- JavaScript - Para la funcionalidad

- API SODA - Fuente de datos del gobierno

- GitHub Pages - Para publicar la web

✨ FUNCIONALIDADES
1. Filtros de búsqueda
- Buscar por texto (cultivo, municipio, departamento)

- Filtrar por departamento

- Filtrar por grupo de cultivo

- Filtrar por cultivo específico

- Filtrar por año

- Filtrar por ciclo (transitorio, permanente, anual)

2. Ordenamiento
- Ordenar por cultivo (A-Z)

- Ordenar por mayor/menor rendimiento

- Ordenar por mayor/menor producción

- Ordenar por mayor área sembrada

- Ordenar por año (reciente a antiguo o viceversa)

3. Visualización de datos
- KPIs (tarjetas con números): total registros, cultivos, departamentos, años

- Chips (resúmenes): producción total, área sembrada, rendimiento promedio, cultivo más productivo, departamento más productivo

- Gráficas de barras: producción por cultivo, grupo, departamento, año, ciclo y rendimiento

- Tabla detallada: todos los registros con paginación (50 por página)

4. Datos que muestra la tabla
- Número de registro

- Departamento

- Municipio

- Grupo de cultivo

- Cultivo

- Nombre científico

- Año

- Período

- Área sembrada (ha)

- Área cosechada (ha)

- Producción (t)

- Rendimiento (t/ha)

- Estado físico

- Ciclo de cultivo
